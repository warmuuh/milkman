# Milkman Mcp Plugin

Support for [Model Context Protocol](https://modelcontextprotocol.io) (MCP) transport

## Features

* support MCP over SSE, Streamable Http and StdIo
* can either connect to remote endpoints or start a local MCP server for interactive testing
* supports calling *Tools*, *Prompts* and *Resources* defined in the MCP server
* Header Support for authentication and custom headers / Environment variables
* Listening to Events returned by the MCP server
